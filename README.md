# Проект с использованием сборщика проектов Webpack.  

  Скомпилированный проект располагается в директории [dist](https://github.com/Tim2104/photography/tree/master/dist)  

## Команды сборки проекта.
Режим Development:  
```
npm run dev
```
Режим Production:
```
npm run prod
```
Запуск сервера:  
```
npm run start
```
